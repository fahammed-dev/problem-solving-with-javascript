## Flipping bits
---
You will be given a list of 32 bit unsigned integers. Flip all the bits ( 1 -> 0 and 0 -> 1 ) and return the result as an unsigned integer.

### Example
---
n = 9<sub>10</sub> <br>
9<sub>10</sub> = 1001<sub>2</sub>
